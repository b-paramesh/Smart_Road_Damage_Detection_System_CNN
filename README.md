# Road Damage Detection using CNN

## Project Overview
CNN-based system to classify:
- Crack
- Pothole
- Manhole

## Features
- Image preprocessing
- Data augmentation
- CNN classification
- Real-time prediction
- Streamlit deployment

## Dataset
Road Damage Dataset (Kaggle)

## Tech Stack
- Python
- TensorFlow
- OpenCV
- Streamlit

## Run

pip install -r requirements.txt

streamlit run app.py
